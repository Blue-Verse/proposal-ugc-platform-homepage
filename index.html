제안 포인트

본 프로젝트는 UGC 커뮤니티 플랫폼과 기업 소개 홈페이지라는 두 축을 하나의 통합 시스템으로 구현하는 프로젝트입니다. 다음과 같은 전략으로 접근합니다:

  • MVP 단계별 납품 — 핵심 기능(UGC 게시판, 회원 시스템, 홈페이지) 우선 개발 후 채팅/고도화 순차 진행으로 리스크 최소화
  • 프로토타입 활용 — 클라이언트가 준비한 바이브코딩 프로토타입을 분석하여 기획 의도를 정확히 반영, 재작업 최소화
  • Next.js + NestJS 풀스택 — 모바일 최적화 웹(PWA)과 반응형 홈페이지를 단일 프레임워크로 구현하여 개발 효율 극대화
  • 유사 프로젝트 경험 — 실시간 채팅, 미디어 공유, 소셜 피드, 관리자 시스템 등 핵심 기능을 다수 프로젝트에서 구현한 경험 보유

예비창업패키지 대응: 사업 계획서에 첨부 가능한 상세 견적서, 공수계산서, PRD, 일정 제안을 본 제안서에 포함하였습니다. 정부지원사업 서류 요구사항에 맞춰 구체적인 산출물과 마일스톤을 제시합니다.

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

당근마켓의 커뮤니티(동네생활 등)와 유사한 UGC 플랫폼을 구축하는 프로젝트입니다. 바이브코딩으로 구체화한 프로토타입을 기반으로 전문적인 UI/UX 디자인과 안정적인 시스템 개발을 목표로 합니다.

  • 사용자가 사진·동영상을 자유롭게 업로드하고 실시간 채팅으로 소통하는 커뮤니티 플랫폼
  • 서비스 신뢰도 확보를 위한 기업 소개 및 정책 안내 홈페이지 통합 구축
  • 예비창업패키지 사업 결과보고에 맞춘 단계별 개발 및 납품

2. 사용자 정의

사용자 유형설명주요 기능
일반 사용자UGC 플랫폼 이용자게시글 작성(사진/동영상), 피드 탐색, 좋아요/댓글, 실시간 채팅, 프로필 관리
관리자플랫폼 운영/관리자회원 관리(제재/탈퇴), 콘텐츠 모니터링(숨김/삭제), FAQ/약관 관리
비회원홈페이지 방문자서비스 소개 열람, FAQ 확인, 회원가입

3. 기능 요구사항

3.1 UGC 게시판 및 미디어 공유

  • [필수] 게시글 작성 — 텍스트 본문 + 사진/동영상 첨부 (다중 파일)
  • [필수] 미디어 최적화 — 이미지 리사이징, 동영상 인코딩/트랜스코딩
  • [필수] 피드 리스트 — 최신순/인기순 정렬, 무한 스크롤
  • [필수] 사용자 인터랙션 — 좋아요, 댓글(작성/삭제), 공유
  • [Phase 2] 해시태그, 검색 기능 고도화

3.2 실시간 텍스트 채팅

  • [필수] 1:1 채팅방 생성 및 실시간 메시지 송수신 (WebSocket)
  • [필수] 채팅 내역 서버 저장 및 스크롤 시 불러오기
  • [필수] 그룹 채팅방 개설
  • [Phase 2] 신규 메시지 푸시 알림 고도화

3.3 사용자 계정 및 프로필

  • [필수] 이메일 회원가입 및 로그인
  • [필수] 소셜 로그인 (Google, Kakao)
  • [필수] 마이페이지 — 닉네임/프로필 이미지 설정, 내 게시글/채팅방 모아보기

3.4 서비스 소개 홈페이지

  • [필수] 서비스/회사 소개 페이지 (비전, 주요 기능, 연혁)
  • [필수] FAQ 게시판 및 1:1 문의 폼
  • [필수] 이용약관 및 개인정보처리방침 페이지

3.5 관리자 시스템 (Admin)

  • [필수] 회원 관리 — 가입자 목록 조회, 불량 이용자 제재/탈퇴 처리
  • [필수] 콘텐츠 관리 — 게시글/미디어 모니터링, 부적절 콘텐츠 숨김/삭제
  • [필수] 사이트 관리 — FAQ/약관 텍스트 등록/수정

4. 비기능 요구사항

5. 기술 제약사항

  • UGC 플랫폼은 모바일 최적화 웹 또는 하이브리드 앱으로 구현 (구체적 방식은 미팅 시 협의)
  • 동영상 인코딩은 서버 측 처리 (클라이언트 부하 최소화)
  • 실시간 채팅은 WebSocket(Socket.io) 기반 — 폴링 방식 불가
  • 소셜 로그인 OAuth 2.0 연동 (Google, Kakao 필수)

6. 범위 외 사항 (Out of Scope)

  • 네이티브 모바일 앱 (iOS/Android) 별도 개발
  • 결제/포인트 시스템
  • AI 기반 콘텐츠 추천 알고리즘
  • 위치 기반 서비스 (GPS 활용)
  • 음성/영상 통화 기능
  • 서버 운영 및 유지보수 (납품 후)
