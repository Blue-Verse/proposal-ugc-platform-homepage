<프로젝트 진행 제안>

■ 프로젝트 분석
당근마켓 커뮤니티(동네생활)와 유사한 UGC 플랫폼을 구축하는 프로젝트로, 사진·동영상 업로드, 실시간 텍스트 채팅, 사용자 인터랙션(좋아요/댓글/공유)이 핵심입니다. 바이브코딩 프로토타입을 기반으로 전문적인 UI/UX 디자인과 안정적인 시스템 개발을 진행하며, 예비창업패키지 사업 일정에 맞춰 MVP 단계별 납품 전략을 제안합니다.

- 기술 스택 제안: Next.js 14 + NestJS 10 + MySQL 8 + Socket.io + AWS S3 + Firebase Auth
- 모바일 최적화 웹(PWA) + 반응형 홈페이지를 단일 프레임워크(Next.js)로 구현하여 개발 효율 극대화
- 프로토타입 분석 → 기획 의도 정확 반영 → 재작업 최소화

■ 작업 일정

[Phase 1: 기획/설계] Day 1-8
- 프로토타입 분석 및 요구사항 정의, 화면 설계(와이어프레임), DB/API 설계
- 산출물: 요구사항 정의서, 와이어프레임, ERD, API 명세서

[Phase 2: UI/UX 디자인] Day 5-15
- UGC 플랫폼 모바일 UI, 서비스 소개 홈페이지 반응형 디자인, 관리자 패널 UI
- 산출물: Figma 디자인 원본 파일

[Phase 3-1: 핵심 기능 개발] Day 12-35
- 회원가입/로그인(소셜 로그인), UGC 게시판(게시글 CRUD, 미디어 업로드, 피드), 기본 관리자
- 산출물: 개발 서버 데모

[Phase 3-2: 부가 기능 개발] Day 30-50
- 실시간 채팅(WebSocket), 좋아요/댓글/공유, 서비스 소개 홈페이지, 관리자 고도화
- 산출물: 스테이징 서버 배포

[Phase 4: QA/배포] Day 46-60
- 통합 테스트, 버그 수정, 성능 최적화, 프로덕션 배포
- 산출물: 테스트 보고서, 소스코드/디자인 이관

■ 마일스톤 및 산출물
- M1 (Day 8): 설계 완료 — 요구사항 정의서, 와이어프레임, ERD 승인
- M2 (Day 15): 디자인 완료 — Figma 디자인 시안 전체 승인
- M3 (Day 35): 핵심 기능 완료 — UGC 게시판, 회원 시스템 데모 확인
- M4 (Day 50): 전체 기능 완료 — 채팅, 홈페이지, 어드민 데모 확인
- M5 (Day 60): 최종 납품 — 프로덕션 배포, 소스코드/디자인 원본 이관
